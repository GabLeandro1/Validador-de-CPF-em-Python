<h2>📌 Descrição do Projeto</h2>

<p>
Este software foi desenvolvido em Python com o objetivo de validar números de CPF informados pelo usuário. O programa utiliza o algoritmo oficial de validação dos dígitos verificadores para identificar se cada CPF é válido ou inválido. Ao final da execução, são exibidas estatísticas contendo a quantidade total de CPFs analisados, bem como o número de registros válidos e inválidos.
</p>
